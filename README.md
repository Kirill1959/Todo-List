# 📝 Todo List — Список дел

[![Maintainability](https://qlty.sh/gh/Kirill1959/projects/Todo-List/maintainability.svg)](https://qlty.sh/gh/Kirill1959/projects/Todo-List)

Простое веб-приложение для управления списком задач, написанное на чистом JavaScript без фреймворков.  
Позволяет добавлять, выполнять и удалять задачи. Все данные сохраняются в браузере через localStorage — задачи не пропадают после перезагрузки страницы.

---

## 🛠 Стек

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Хранилище:** localStorage (браузер)
- **Backend / DB:** не используется

---

## ⚙️ Как запустить локально

1. Склонируй репозиторий:
   ```bash
   git clone https://github.com/Kirill1959/Todo-List.git
   ```
2. Открой папку проекта и запусти `index.html` в браузере  
   (или через расширение **Live Server** в VS Code)

## ✅ Функциональность

| Функция              | Описание                                         |
| -------------------- | ------------------------------------------------ |
| Добавление задачи    | Введи текст и нажми «Добавить» или Enter         |
| Отметить выполненной | Нажми на чекбокс рядом с задачей                 |
| Удалить задачу       | Нажми кнопку ✕ справа                            |
| Фильтрация           | Переключай вкладки: Все / Активные / Выполненные |
| Очистить выполненные | Кнопка внизу удаляет все выполненные задачи      |
| Сохранение           | Задачи сохраняются в localStorage                |
